1 BUSINESS REQUIREMENTS (БІЗНЕС-ВИМОГИ)
1.1 Background (Передумови)

У сучасному світі догляд за домашніми тваринами стає все більш популярним, а відповідальне розведення та знаходження ідеальних партнерів для в’язки є ключовими аспектами для забезпечення здоров’я та благополуччя тварин. В цьому контексті, програмна система пошуку партнера для в’язки виступає як необхідна передумова для створення оптимальних пар, що сприяє покращенню порід та здоров’я наступних поколінь.
Перш за все, власники тварин стикаються з викликом знаходження підходящих партнерів, які б відповідали певним критеріям, таким як порода, вік, здоров’я та характер. Система пошуку повинна здійснювати постійне відстеження та аналіз потенційних кандидатів, щоб забезпечити, що кожна тварина знайде найбільш підходящого партнера для в’язки.
Крім того, генетичне здоров’я та спадковість є ще одними факторами, які мають значний вплив на успішність в’язки. Різні породи тварин мають різні генетичні особливості та потреби. Програмна система пошуку повинна забезпечити точний аналіз генетичних даних та історії здоров’я, дозволяючи власникам вибирати найкращих партнерів для своїх улюбленців.
Соціальна сумісність та поведінка тварин є важливими складовими для забезпечення успішної в’язки. Система повинна бути здатна аналізувати поведінкові моделі та соціальні взаємодії між тваринами, забезпечуючи вибір партнерів, які будуть добре ладнати один з одним.
Таким чином, створення програмної системи пошуку партнера для в’язки є необхідним для власників тварин, які прагнуть забезпечити здоров’я та благополуччя своїх улюбленців, а також покращити породи та здоров’я наступних поколінь.

1.2 Business Opportunities (Бізнес-можливості)

Аналоги:
-	Lapki.pet – це онлайн-платформа, яка спеціалізується на пошуку партнерів для в’язки домашніх тварин. Вона пропонує користувачам можливість розміщувати оголошення та шукати партнерів за різними критеріями, такими як порода, вік, та інші характеристики. Основні переваги сервісу включають зручний інтерфейс та велику базу даних, що дозволяє власникам тварин знаходити потенційних партнерів для своїх улюбленців. Однак, сервіс може мати обмеження у відсутності глибокого аналізу генетичної сумісності, що є важливим аспектом при виборі партнера для в’язки, щоб уникнути спадкових захворювань та покращити породу.
-	Murkote.com – це інформаційний ресурс, який надає поради та рекомендації щодо в’язки породистих котів. Він містить корисну інформацію про підготовку до в’язки, вибір партнера, та репродуктивне здоров’я котів. Murkote.com може бути цінним джерелом для власників котів, які шукають поради щодо розведення породистих котів, але, на відміну від Lapki.pet, не пропонує прямого сервісу для знаходження партнерів для в’язки.
-	Наша програмна система має потенціал заповнити прогалини, які існують у цих сервісах, пропонуючи інтегрований підхід до пошуку партнерів для в’язки з використанням передових технологій для аналізу генетичної сумісності та поведінкових характеристик. Це може включати розширену базу даних з детальними профілями тварин, інтеграцію з ветеринарними базами даних для перевірки здоров’я та вакцинації, а також інструменти для спілкування між власниками тварин. Основний спосіб монетизації може включати підписку для доступу до преміум-функцій, партнерство з ветеринарними клініками та рекламу товарів для тварин. Такий підхід дозволить створити всебічний сервіс, який буде корисним як для власників домашніх тварин, так і для професіоналів у сфері розведення тварин.

2 VISION OF THE SOLUTION (КОНЦЕПЦІЯ РІШЕННЯ)
2.1 Vision Statment (Окреслення концепції)

Наш продукт є важливим для забезпечення власників тварин інструментом для точного пошуку партнерів для в’язки. Впровадження даної програмної системи дозволить користувачам ефективно вибирати партнерів для своїх тварин, забезпечуючи оптимальні умови для розведення та здоров’я нащадків.
Основні переваги цього продукту полягають у зменшенні ризиків, пов’язаних з вибором непідходящих партнерів для в’язки, та підвищенні якості породи. Програмна система надає автоматизований механізм, який враховує генетичну сумісність, породу, вік та інші важливі характеристики, що дозволяє користувачам знаходити найкращих партнерів для своїх тварин.
Впровадження програмної системи пошуку партнера для в’язки допоможе власникам тварин досягти оптимальних результатів у розведенні. Вони зможуть забезпечити відповідальний вибір партнерів, покращити здоров’я та характеристики нащадків, а також збільшити свої шанси на успіх у виставках та змаганнях. Крім того, це дозволить їм бути більш успішними на ринку, привертати нових клієнтів та збільшувати свої доходи.
Програмна система пошуку партнера для в’язки є інноваційним рішенням, яке допоможе оптимізувати процес вибору партнерів для в’язки та підвищити ефективність розведення тварин. Завдяки використанню передових технологій та алгоритмів, наш продукт стане незамінним помічником для всіх, хто прагне до відповідального та якісного розведення тварин.

2.2 Major Features (Головна функціональність)

MF-1: Реєстрація та авторизація користувачів. Можливість створення особистого профілю для кожного користувача з детальною інформацією про їхніх тварин

MF-4: Історія в’язок. Зберігання та відстеження історії в’язок та нащадків для кожної тварини
MF-5: Налаштування параметрів пошуку. Можливість налаштування параметрів пошуку для відповідності специфічним вимогам користувача
MF-6: Сповіщення та нотифікації. Автоматичні сповіщення про нових потенційних партнерів та нагадування про важливі дати в’язок
MF-7: Інтеграція з ветеринарними базами даних. Інтеграція з ветеринарними базами даних для перевірки здоров’я та вакцинацій тварин
MF-8: Комунікаційні інструменти. Інструменти для спілкування між власниками тварин для обговорення деталей в’язки
MF-9: Мобільний додаток та веб-інтерфейс. Можливість керування профілем та пошуком партнерів через мобільний додаток або веб-інтерфейс
MF-10: Технічна підтримка. Підтримка користувачів з будь-якими технічними питаннями або проблемами, пов’язаними з використанням системи

3 SCOPE AND LIMITATIONS (РАМКИ ТА ОБМЕЖЕННЯ)
3.1 Scope of Initial Release (Рамки первинного випуску)

У веб-застосунку користувачі будуть мати можливість:
-	реєструватися та авторизуватися;
-	переглядати профілі потенційних партнерів для в’язки;
-	отримувати сповіщення про нових партнерів;
- налаштовувати параметри пошуку відповідно до вимог своїх тварин;
- взаємодіяти з іншими користувачами через вбудовані комунікаційні інструменти.
До функціоналу IoT відноситься:
- інтеграція з ветеринарними базами даних для перевірки здоров’я тварин.
У мобільному застосунку користувачі будуть мати можливість:
- реєструватися та авторизуватися;
- переглядати профілі та історію в’язок своїх тварин;
- отримувати сповіщення про важливі події пов’язані з в’язкою;
- налаштовувати параметри системи для оптимізації пошуку партнерів.
MF-1: Реєстрація та авторизація 
Користувачі зможуть створити та увійти до свого акаунту, використовуючи логін та пароль.
MF-2: Пошук та перегляд партнерів для в’язки
Користувачі зможуть переглядати детальну інформацію про потенційних партнерів для своїх тварин.
MF-3: Сповіщення та нотифікації
Система надаватиме користувачам сповіщення про нових партнерів та інші важливі події.
MF-4: Комунікаційні інструменти
Користувачі зможуть спілкуватися з іншими власниками тварин для обговорення деталей в’язки.
MF-5: Інтеграція з ветеринарними базами даних
Система інтегруватиметься з ветеринарними базами даних для перевірки здоров’я та вакцинацій тварин.
MF-6: Мобільний додаток та веб-інтерфейс
Користувачі зможуть використовувати мобільний додаток та веб-інтерфейс для керування своїм профілем та пошуком партнерів.
MF-7: Технічна підтримка
Користувачі отримуватимуть технічну підтримку для вирішення будь-яких питань, пов’язаних з використанням системи.

3.2 Scope of Subsequent Releases (Рамки наступних випусків) 

У веб-застосунку користувачі будуть мати можливість: 
- автоматично вибирати партнерів для в’язки на основі рекомендацій системи;
- встановлювати параметри пошуку для оптимального відбору партнерів;
- відстежувати успіх попередніх в’язок та їх вплив на генетичну лінію;
- керувати системою через веб-застосунок для зручності та ефективності.
Адміністратор у веб-застосунку буде мати можливість:
- відповідати на запити техпідтримки та надавати консультації користувачам.
У мобайл-застосунку користувачі також будуть мати можливість:
- автоматично налаштовувати параметри пошуку відповідно до оновлень системи;
- встановлювати специфічні параметри для кожної породи тварин;
- відстежувати історію в’язок та аналізувати їх результати;
- керувати системою через мобільний додаток для зручності використання в будь-якому місці.
MF-4: Розширений пошук та відбір партнерів
Система забезпечить автоматичний аналіз сумісності та рекомендації для вибору партнерів.
MF-6: Інтегровані комунікаційні засоби
Система включатиме інтегровані засоби для спілкування між користувачами, щоб обговорювати деталі в’язки.
MF-8: Аналіз генетичних ліній
Система надаватиме інструменти для аналізу генетичних ліній та оцінки їх впливу на здоров’я та характеристики нащадків.
MF-9: Дистанційне керування та моніторинг
Користувачі матимуть можливість дистанційно керувати своїми профілями та відстежувати статус пошуку партнерів.
MF-10: Розширена техпідтримка
Користувачі отримають доступ до розширеної техпідтримки, яка допоможе їм вирішувати технічні питання та оптимізувати використання системи.

4 BUSINESS CONTEXT (БІЗНЕС-КОНТЕКСТ)
4.3 Operating Environment (Робоче середовище)

Даний продукт буде використовуватися через браузер на пк або за допомогою телефону та планшету. Для використання мобільного додатку має використовуватися система Android 12 та вище або iOS 15 та вище.
Інформація про тварин та користувачів буде зберігатися в базі даних SQLite. 
Для розробки веб-застосунку буде використовуватися React, bootstrap 5. 
Для розробки серверної частини продукту буде використовуватися Django.
Для розробки IoT частини продукту буде використовуватися Arduino. Середою розробки буде Arduino IDE. Прилад буде складатися з модулів GPS для визначення місцезнаходження тварин, модулів зв’язку для передачі даних про потенційних партнерів та wi-fi модулю для підключення до інтернету та обміну даними з центральною базою даних системи.
Для розробки мобільного застосунку буде використовуватися Kotlin та Android Studio.
